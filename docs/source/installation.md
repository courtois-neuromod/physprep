# Installation

## Cloning the project from Github

1. Clone the project:

```bash
git clone git@github.com:courtois-neuromod/physprep.git
```

2. Create a virtual environment:

```bash
python3 -m venv <venv_name>
source <venv_name>/bin/activate
```

3. Install the dependency:

```bash
cd physprep
pip install .
```
